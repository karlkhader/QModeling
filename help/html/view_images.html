How can I see the parametric images obtained?

Once you have calculated these images, the View Parametric Images panel will be enabled. Select the image from the drop-down list and press the View Image button. The image will be shown in a new window.

Can I see more than one image at the same time?

Yes. Each time you push the View Image button, the image selected from the drop-down list will be displayed in a new window.

Can I visualize images stored in my computer?

Yes. The option View Image is shown on the QModeling tab. Select it to view any NIfTI or Analyze 3D image you have on your computer.
